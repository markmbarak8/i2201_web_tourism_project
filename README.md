<h1>MyTrip (Web tourism project for I2201)</h1>

<p>A web developement project utilizing the three fundamental tools of web developement,html,CSS and Javascript.</p>

<h2>About this project</h2>
<p>The aim of this university project is to create a website for tourrist attraction to the most famous monuments around the world.We included a home page as an introduction to users to the site, a gallery page that displays the monuments one by one, a feedback page with with interactive user interface for submitting a feedback and for each of the six monuments we created it's own dedicated page that describes it very well with images and historical data.</p>
<h2>Tools used and notable achievements</h2>
<p>We obviously worked with vanilla html,CSS and javascript without any frameworks since the aim of this project is to test our skills in web. We did some notable achivements in our project that are worth mentioning: </p>

<dl>
<dt>Carousel images.</dt>
<dd>We created carosel images in the home page using translation and various methods to create a beautiful and smooth animation of the images.</dd>
<dt>Feedback Form.</dt>
<dd>We created a responsive feedback form in the feedback page with a beautiful background image.</dd>
<dt>Sticky images</dt>
<dd>Making the images stick in the home page while scrolling was small but astounding for us.</dd>
